Repository:

https://github.com/sabirama/assets/tree/master

Put into your html file <link rel="stylesheet" href="https://sabirama.github.io/assets/icons.css" >

or the minified version <link rel="stylesheet" href="https://sabirama.github.io/assets/icons-mini.css" >


Create an element with i tag. There are three types of classes in thicons project: size, color and i-name which you can use to edit your element by adding those to the classes.

sample:

<i class="{icon-name} {size} {color} hover-{color}"></i>

Available sizes:

  • default = 24px height
  • s-icon = 16px height
  • m-icon = 32px height
  • l-icon = 40px height
  • xl-icon = 48px height

Colors:

white pink red green forest cyan blue navy yellow gold orange purple indigo brown gray silver

Change color when hovering over an icon by adding a hover-{color of choice} into the class.

You can create a new color class using the css filter property. Try going to https://codepen.io/sosuke/pen/Pjoqqp and put the hexcode of the color of your choice then take the filter into your color class.

NOTE: it icons better to use the color type only to black icons.
