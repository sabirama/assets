Create an element with i tag. There are three types of classes for the icons: size, color and icon-name which you can use to edit your element by adding them to the element class.

sample:

<i class="icon {icon-name} {size} {color} hover-{color}"></i>

Available sizes:

  • default = 24px height
  • s-icon = 16px height
  • m-icon = 32px height
  • l-icon = 40px height
  • xl-icon = 48px height

Colors:

white pink red green forest cyan blue navy yellow gold orange purple indigo brown gray silver

Change color when hovering over an icon by adding a hover-{color of choice} into the class.

You can create a new color class using the css filter property. Try going to https://codepen.io/sosuke/pen/Pjoqqp and put the hexcode of the color of your choice then take the filter into your color class.

NOTE: it icons better to use the color type only to black icons.

CSS ANIMATION

Animation have three class types: animate, delay and animation-name.

NOTE: add overflow hidden to parent element of animation element if you don't want the whole page to see it during animation.

Duration

Add {animate} to class to add animation to html element.

  • default duration is 1s.
  • animate-halfs = .5s
  • animate-1halfs = 1.5s
  • animate-2s = 2s
  • animate-2halfs = 2.5s
  • animate-3s = 3s
  • animate-3halfs = 3.5s
  • animate-4s = 4s
  • animate-4halfs = 5.5s
  • animate-5s = 5s
  • animate-5halfs = 5.5s/li>

Delay

Add {delay} to class add delay before animation starts.

If delay is not added the animation starts right away

  • default when added a delay is 1s.
  • delay-halfs = .5s
  • delay-1halfs = 1.5s
  • delay-2s = 2s
  • delay-2halfs = 2.5s
  • delay-3s = 3s
  • delay-3halfs = 3.5s
  • delay-4s = 4s
  • delay-4halfs = 4.5s
  • delay-5s = 5s
  • delay-5halfs = 5.5s

Date

Add date to your website easily with custom classes.

Start by adding "date" class for dates and "time" class for time. "date" by default gets the year-month-day and "time" by default gets hour-minute-second.

There is also time-12 which give a 12 hour format of hours

Or add "year" or "month" or "day" or "month-name" or "day-name" after date.

example: <p class="date year"></p> this will show the current year: .

As for time, add "hour" or "hour-12", or "minute" or "second" or "meridian" after time class.

exmple: <p class="time meridian"></p> this will show if it is am or pm: .
